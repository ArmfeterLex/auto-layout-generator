# Auto Layout Generator

Приложение на C# Windows Forms для автоматического создания элементов управления в строку с переносом.

## Функциональность

### ➕ Создание элементов (button1)
- Случайный выбор типа:
  - Кнопка (100×30) — с текстом "Кнопка"
  - Многострочное текстовое поле (150×30)
- Автоматическая расстановка в строку с отступом 10px
- Перенос на новую строку при достижении правой границы формы

### 📏 Увеличение полей (button2)
- Все существующие текстовые поля увеличивают высоту в 2 раза

## Интерфейс

- button1 — создать случайный элемент
- button2 — увеличить высоту всех текстовых полей
- Область формы — отображение созданных элементов

## Пример работы

1. Нажатие button1 → создаётся кнопка в (20,60)
2. Нажатие button1 → создаётся поле в (130,60)
3. Нажатие button1 → создаётся кнопка в (290,60) (если помещается)
4. Нажатие button2 → все поля увеличиваются вдвое
5. При достижении правой границы — перенос на следующую строку

## Требования

- Windows
- .NET Framework / .NET Core с WinForms

## Примечание

Текстовые поля создаются с Multiline = true. При переносе строки отступ по Y увеличивается на 50px. Формула переноса работает только при добавлении новых элементов.
